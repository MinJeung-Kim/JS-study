# SASS(SCSS)

## SASS란?

***

* Syntactically Awesome StyleSheets
* CSS 전처리기(Preprocessor), 이를 위한 구문 이다.
* CSS의 단점을 보완. CSS 확장 언어다.
* SCSS는 SASS언어의 하위 개념이다.
