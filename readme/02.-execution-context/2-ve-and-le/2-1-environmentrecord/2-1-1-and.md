# 2-1-1) 함수 선언문 & 함수 표현식

### 함수 정의 3가지방식

```javascript
function a () { /* ... */ };
a();

var b = function() { /* ... */ };
b();

var c = function d() { /* ... */ };
c();
d(); // ⚠️error
```

### 함수 선언문과 함수 표현식 차이점

```javascript
console.log(sum(1, 2)); // 3
console.log(multiply(3, 4)); // ⚠️TypeError: multiply is not a function

// 선언문
function sum(a, b) {
  return a + b;
}

// 표현식
var multiply = function (a, b) {
  return a * b;
};

// ===================================[ 호이스팅 ]====================================


```
