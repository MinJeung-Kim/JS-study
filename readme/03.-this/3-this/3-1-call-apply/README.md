# 3-1) Call / Apply 메서드

* **this를 예측하기 어렵게 만들어** **코드 해석을 방해**한다는 단점.
* **ES5 이하의 환경에서는 마땅한 대안이 없기 때문**에 실무에서 매우 광범위하게 활용됨.
